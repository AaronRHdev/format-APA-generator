# Format APA Generator

## Descripción

Este proyecto es una herramienta que genera automáticamente citas y referencias en formato APA. Es útil para estudiantes, investigadores y profesionales que necesitan crear referencias de manera rápida y precisa.

## Instalación

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com/AaronRHdev/format-APA-generator.git
   ```
2. Abre el archivo `index.html` directamente en tu navegador para utilizar la herramienta, o si lo prefieres puedes acceder al siguiente link: https://aaronrhdev.github.io/format-APA-generator/

   > Nota: Asegúrate de tener un navegador actualizado para evitar problemas de compatibilidad con las tecnologías empleadas.

## Uso

El Format APA Generator es muy fácil de usar:

1. Abre el archivo `index.html` en tu navegador o accede a: https://aaronrhdev.github.io/format-APA-generator/
   - selecciona la referencia que deseas generar.
   <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/menu.gif" align="center" width="600"></picture>

3. Completa el formulario con la información requerida, como el nombre del autor, título, fecha de publicación, etc.
     <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/inputs.gif" align="center" width="600"></picture>
     
4. Haz clic en el botón "Generar cita" para que la cita en formato APA aparezca en la parte inferior de la pantalla.
   - Copia la cita generada y pégala donde la necesites.
   <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/generate.gif" align="center" width="600"></picture>

#### Otras funciones

- Botón limpiar: Lo puedes usar para limpiar el formulario.
  <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/clean.gif" align="center" width="600"></picture>
- Botón agregar: Esta representado con un icono para agregar personas, te permite agregar una persona más para referirla como autor o presentador.
     - Puedes eliminar la nueva persona agregada con el botón limpiar.
     <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/add.gif" align="center" width="600"></picture>
- Botón inicio: te permite volver a la ventana inicial.
  <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/home.gif" align="center" width="600"></picture>

> Nota: El generador mostrará un mensaje de error en caso de no haber llenado los datos correctamente.
   <br><br><picture><img src="https://github.com/AaronRHdev/format-APA-generator/blob/main/assets/img/screenshots/.gif/arror.gif" align="center" width="600"></picture>

El generador está diseñado para manejar entradas de diferentes tipos de fuentes. Puedes seleccionar entre varias opciones, como libros, artículos o sitios web, según el tipo de referencia que quieras crear.
